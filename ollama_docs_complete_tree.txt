    |-- /modelfile
    |   |-- Format: INSTRUCTION arguments (not case-sensitive)
    |   |-- Instructions:
    |   |   |-- FROM (required) -- existing model / Safetensors dir / GGUF file
    |   |   |-- PARAMETER -- runtime params:
    |   |   |   |-- num_ctx (default 2048)
    |   |   |   |-- temperature (default 0.8)
    |   |   |   |-- top_k (default 40)
    |   |   |   |-- top_p (default 0.9)
    |   |   |   |-- min_p (default 0.0)
    |   |   |   |-- repeat_penalty (default 1.1)
    |   |   |   |-- repeat_last_n (default 64)
    |   |   |   |-- num_predict (default -1, infinite)
    |   |   |   |-- draft_num_predict (default 4)
    |   |   |   |-- seed (default 0)
    |   |   |   |-- stop -- stop sequences
    |   |   |-- TEMPLATE -- Go template syntax
    |   |   |   |-- {{ .System }}
    |   |   |   |-- {{ .Prompt }}
    |   |   |   |-- {{ .Response }}
    |   |   |-- SYSTEM -- system message
    |   |   |-- ADAPTER -- LoRA adapters (Safetensors or GGUF)
    |   |   |-- LICENSE -- legal license text
    |   |   |-- MESSAGE -- example conversation history
    |   |   |   |-- system
    |   |   |   |-- user
    |   |   |   |-- assistant
    |   |   |-- REQUIRES -- minimum Ollama version
    |   |-- Examples: basic Modelfile, ollama create, ollama show --modelfile

ENVIRONMENT VARIABLES REFERENCE
================================================================================

| Variable                    | Purpose                                      |
|-----------------------------|----------------------------------------------|
| OLLAMA_HOST                 | Bind address (default 127.0.0.1:11434)       |
| OLLAMA_MODELS               | Model storage directory                      |
| OLLAMA_CONTEXT_LENGTH       | Default context window size                  |
| OLLAMA_KEEP_ALIVE           | Model memory retention duration              |
| OLLAMA_MAX_QUEUE            | Max queued requests (default 512)            |
| OLLAMA_MAX_LOADED_MODELS    | Concurrent models (default 3xGPU count)      |
| OLLAMA_NUM_PARALLEL         | Parallel requests per model (default 1)      |
| OLLAMA_FLASH_ATTENTION      | Force Flash Attention (1=on, 0=off)          |
| OLLAMA_KV_CACHE_TYPE        | K/V quant: f16, q8_0, q4_0                   |
| OLLAMA_LLM_LIBRARY          | Override autodetected LLM library            |
| OLLAMA_ORIGINS              | Allowed CORS origins                         |
| OLLAMA_API_KEY              | Cloud API authentication                     |
| OLLAMA_NO_CLOUD             | Disable cloud features (1=disabled)          |
| OLLAMA_DEBUG                | Enable debug logging (1=enabled)             |
| OLLAMA_TMPDIR               | Temporary executable directory               |
| HTTPS_PROXY                 | Outbound proxy for model pulls               |
| CUDA_VISIBLE_DEVICES        | NVIDIA GPU selection                         |
| ROCR_VISIBLE_DEVICES        | AMD GPU selection                            |
| HSA_OVERRIDE_GFX_VERSION    | AMD GPU target override                      |
| GGML_VK_VISIBLE_DEVICES     | Vulkan GPU selection                         |
| OLLAMA_VULKAN               | Disable Vulkan (0=disabled)                  |
